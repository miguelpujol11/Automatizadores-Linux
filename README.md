
# Bash

Estos proyectos de BASH esta echo para la ayuda de la automatizacion de principiantes de linux.



## Installation



La instalacion con GIT incluye "automatizador.sh y nmap.sh" 


```
1) git clone https://github.com/miguelpujol11/Automatizadores-Linux.git
2) cd Automatizadores-Linux
3) chmod 777 Automatizador.sh o nmap.sh
4) ejecuta uno de los scripts que quieras usar con ./Automatizador.sh o ./nmap.sh
